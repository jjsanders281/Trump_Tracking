# Shared Reference: Source Quality Matrix

## Tier Definitions
- Tier 1: primary records and top reliability sources.
  - Examples: full transcript/video, AP/Reuters, court and government records.
- Tier 2: major outlets with editorial controls.
  - Examples: major newspapers and broadcast outlets.
- Tier 3: partisan or low-transparency outlets.
  - Use as context only; not sufficient for verdict assignment.

## Required Evidence Rules
- Every claim requires one primary source.
- `false`, `misleading`, and `contradicted` verdicts require a clear evidence chain.
- If primary evidence is weak or missing context, keep status `pending`.

## Publication Gate
- Default public list includes `verified` only.
- Any unresolved source conflict blocks publication.
