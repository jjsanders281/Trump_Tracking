# Editorial Reference

## Review Checklist
- Is wording neutral and precise?
- Is claim extraction truly atomic?
- Does rationale map to evidence without leaps?
- Are topic and impact score consistent with taxonomy?

## Approval Rules
- Approve only if source trail is complete.
- Reject if language overstates what evidence proves.
- Request revision when rationale is correct but unclear.

## Corrections
- Preserve prior state in revision history.
- Publish correction notes for substantive changes.
