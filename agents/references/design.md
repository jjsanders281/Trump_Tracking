# Design Reference

## Core UX Goals
- Make source links and verdict rationale immediately visible.
- Reduce time to find a claim by date, topic, or quote text.
- Support both quick scanning and deep research paths.

## Priority Interfaces
- Timeline view
- Filter panels (topic, verdict, date, impact)
- Claim detail with source stack and revision history
- Contradiction link view

## Accessibility Baseline
- Keyboard navigation for search and results table
- Clear focus states
- Color contrast that does not encode meaning alone
