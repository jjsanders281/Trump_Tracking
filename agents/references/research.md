# Research Reference

## Intake Standard
Capture this minimum packet for every candidate:
- Exact quote text
- Datetime and venue
- Primary source URL
- Proposed claim text (atomic)
- Topic and impact score proposal

## Disambiguation Heuristics
- If one quote contains multiple assertions, split into multiple claims.
- If quote timing is uncertain, hold until timestamp confidence is high.
- If source media is clipped, locate full context before intake.

## Archive Discipline
- Save stable links when possible.
- Prefer first-publication records for chronology.
