# Fact-Check Reference

## Verdict Mapping
- `true`: evidence aligns materially with claim.
- `mixed`: claim includes both accurate and inaccurate elements.
- `misleading`: technically partial but omits context that changes interpretation.
- `false`: evidence materially contradicts claim.
- `unverified`: evidence insufficient.
- `unfulfilled`: promised action did not occur in stated timeframe.
- `contradicted`: later claim conflicts with prior documented claim.

## Evidence Chain Pattern
1. Anchor to primary source.
2. Add corroborating source(s) by tier.
3. Resolve timeline.
4. Assign verdict with uncertainty statement.

## Stop Conditions
- Primary source unavailable.
- Source conflict not resolvable with current evidence.
