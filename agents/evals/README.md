# Agent Evaluations

Use this folder to score each role against repeatable scenarios.

## Process
1. Select a role folder.
2. Run all cases in `cases.jsonl`.
3. Grade with `rubric.md`.
4. Record score in `agents/scorecards/weekly-template.md`.
5. Open action items for any criterion scoring below target.

## Scoring Scale
- 0: failed
- 1: partially met
- 2: met

Target: average >= 1.7 per criterion per role.
